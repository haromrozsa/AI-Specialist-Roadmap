# How to Capture MLflow UI Screenshots

## Step 1: Start MLflow UI

```bash
mlflow ui
```

Open browser to: http://localhost:5000

## Step 2: Recommended Screenshots

### 1. Experiments List View
- Shows all 5 runs with parameters and metrics
- File: `screenshots/experiments_list.png`

### 2. Run Comparison View
- Select 2-3 runs and click "Compare"
- Shows side-by-side parameter and metric comparison
- File: `screenshots/run_comparison.png`

### 3. Single Run Detail View
- Click on any run to see full details
- Shows all logged params, metrics, and artifacts
- File: `screenshots/run_detail.png`

### 4. Metrics Chart (Optional)
- Click on a metric to see visualization
- File: `screenshots/metrics_chart.png`

## Note

Screenshots should be placed in the `screenshots/` directory for GitHub proof as per Week 21 specification.
